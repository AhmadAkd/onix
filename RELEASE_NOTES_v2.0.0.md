## üöÄ Getting Started

### Prerequisites
- Python 3.9+ (Python 3.12 recommended)
- pip (Python package installer)
- Git (for development)

### Installation
```bash
# Clone the repository
git clone https://github.com/AhmadAkd/onix-client.git
cd onix-client

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py
```

### Building Executable
```bash
# Windows
build.bat

# macOS/Linux
chmod +x build.sh
./build.sh
```
