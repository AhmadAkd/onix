## ✨ 什么让 Onix 与众不同？

Onix 不仅仅是一个代理客户端——它是一个**综合网络解决方案**，将尖端技术与直观的用户体验相结合。使用 `PySide6` 构建，由最新的 `sing-box` 核心驱动，Onix 在美观、易用的界面中提供企业级功能。

### 🎯 **为什么选择 Onix？**

- 🔥 **下一代性能**: 实时速度测试、高级健康监控和智能故障转移
- 🛡️ **军用级安全**: Kill Switch、DNS 泄漏保护和全面的隐私控制
- 🎨 **令人惊叹的 UI/UX**: 6+ 自定义主题、RTL 支持和响应式设计
- 🌐 **通用兼容性**: 支持所有主要代理协议和平台
- 📊 **高级分析**: 实时统计、性能监控和详细诊断
- 🔧 **开发者友好**: 开源、文档完善且高度可扩展

## 🚀 快速开始

### 📋 **先决条件**

- **Python 3.9+** (推荐 Python 3.12)
- **pip** (Python 包安装程序)
- **Git** (用于开发)

### ⚡ **安装**

```bash
# 克隆仓库
git clone https://github.com/AhmadAkd/onix-client.git
cd onix-client

# 安装依赖
pip install -r requirements.txt

# 运行应用程序
python main.py
```

### 🏗️ **构建可执行文件**

```bash
# Windows
build.bat

# macOS/Linux
chmod +x build.sh
./build.sh

# 或使用通用构建脚本
python build_all.py
```

## 🎨 用户界面

### 🖥️ **主界面**

Onix 具有**现代、直观的界面**，包含四个主要部分：

#### 🔌 **连接视图**

- **服务器管理**: 添加、编辑和组织您的代理服务器
- **实时状态**: 实时连接状态和性能指标
- **快速操作**: 一键连接、测试和管理服务器

#### 🛣️ **路由视图**

- **智能规则**: 高级路由配置
- **绕过列表**: 自定义域名和 IP 绕过规则
- **流量控制**: 精细的流量管理

#### 📋 **日志视图**

- **实时监控**: 实时应用程序和核心日志
- **高级过滤**: 按级别、来源或内容搜索和过滤
- **导出功能**: 保存日志进行分析

#### ⚙️ **设置视图**

- **外观**: 主题、颜色和显示选项
- **网络**: DNS、绕过规则和连接设置
- **安全**: Kill switch、DNS 保护和隐私控制
- **高级**: 性能调整和开发者选项

## 🔧 配置

### ⚙️ **网络设置**

```yaml
# 示例配置
network:
  mode: "rule-based"  # 或 "global"
  dns_servers: ["1.1.1.1", "8.8.8.8"]
  bypass_domains: ["*.local", "*.lan"]
  tun_mode: true
  hysteria2_bandwidth:
    upload: "100 Mbps"
    download: "500 Mbps"
```

### 🛡️ **安全设置**

```yaml
security:
  kill_switch: true
  dns_leak_protection: true
  firewall_integration: true
  privacy_mode: "strict"
  audit_logging: true
```
