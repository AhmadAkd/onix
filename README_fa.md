## ✨ چه چیزی Onix را خاص می‌کند؟

Onix فقط یک کلاینت پروکسی دیگر نیست – این یک **راه‌حل جامع شبکه‌ای** است که تکنولوژی‌های پیشرفته را با تجربه کاربری بصری ترکیب می‌کند. ساخته شده با `PySide6` و قدرت گرفته از آخرین هسته `sing-box`، Onix امکانات درجه سازمانی را در یک رابط زیبا و قابل دسترس ارائه می‌دهد.

### 🎯 **چرا Onix را انتخاب کنیم؟**

- 🔥 **عملکرد نسل بعدی**: تست سرعت بلادرنگ، نظارت پیشرفته سلامت و failover هوشمند
- 🛡️ **امنیت درجه نظامی**: Kill Switch، محافظت از نشت DNS و کنترل‌های جامع حریم خصوصی
- 🎨 **UI/UX خیره‌کننده**: ۶+ تم سفارشی، پشتیبانی RTL و طراحی responsive
- 🌐 **سازگاری جهانی**: پشتیبانی از تمام پروتکل‌های پروکسی اصلی و پلتفرم‌ها
- 📊 **تحلیل‌های پیشرفته**: آمار بلادرنگ، نظارت عملکرد و تشخیص‌های تفصیلی
- 🔧 **دوستدار توسعه‌دهنده**: منبع باز، مستندسازی کامل و بسیار قابل توسعه

## 🚀 امکانات اصلی

### 🌐 **پشتیبانی جهانی پروتکل‌ها**

| **پروتکل‌های استاندارد** | **پروتکل‌های مدرن** | **تخصصی** |
|------------------------|---------------------|-----------------|
| ✅ VLESS | ✅ TUIC v5 | ✅ WireGuard |
| ✅ VMess | ✅ Hysteria2 | ✅ SSH |
| ✅ Shadowsocks | ✅ Reality | ✅ Custom |
| ✅ Trojan | ✅ HTTP/2 | ✅ SOCKS5 |

### 🖥️ **مدیریت هوشمند سرور**

- **📡 کشف هوشمند**: تشخیص خودکار سرورها از QR کد، لینک اشتراک یا ورودی دستی
- **📁 سازماندهی پیشرفته**: گروه‌بندی سرورها بر اساس مکان، ارائه‌دهنده یا دسته‌بندی سفارشی
- **🔄 آپدیت بلادرنگ**: مدیریت اشتراک زنده با آپدیت فوری سرورها
- **📊 تحلیل عملکرد**: متریک‌های تفصیلی ping، سرعت و قابلیت اطمینان
- **🎯 Auto-Failover**: تعویض هوشمند سرور در صورت مشکل اتصال

### 🛡️ **امکانات امنیتی سازمانی**

- **⚡ Kill Switch**: قطع فوری اینترنت در صورت خرابی پروکسی
- **🔒 محافظت از نشت DNS**: نظارت و اصلاح پیشرفته DNS
- **🔐 کنترل‌های حریم خصوصی**: محافظت جامع داده و ناشناس‌سازی
- **🛡️ یکپارچگی فایروال**: اجرای امنیت در سطح سیستم
- **📋 ثبت حسابرسی**: ردیابی و گزارش‌دهی کامل فعالیت

### 🎨 **UI زیبا و قابل سفارشی‌سازی**

- **🌈 ۶+ تم پریمیوم**: از حالت‌های تاریک ظریف تا استایل‌های نئون زنده
- **🌍 چندزبانه**: پشتیبانی کامل RTL برای فارسی، عربی و بیشتر
- **📱 طراحی Responsive**: تطبیق کامل با هر اندازه صفحه
- **⌨️ میانبرهای صفحه‌کلید**: امکانات کارایی برای کاربران حرفه‌ای
- **🎛️ تنظیمات پیشرفته**: کنترل دقیق بر هر جنبه

### 📊 **نظارت و تحلیل بلادرنگ**

- **📈 تست سرعت زنده**: اندازه‌گیری پهنای باند بلادرنگ
- **💓 نظارت سلامت**: ارزیابی مداوم سلامت سرور
- **📊 متریک‌های عملکرد**: آمار تفصیلی و تحلیل روند
- **🔍 تشخیص شبکه**: ابزارهای جامع عیب‌یابی
- **📋 گزارش‌های صادراتی**: قابلیت‌های گزارش‌دهی درجه حرفه‌ای

- **📡 مدیریت اشتراک**: آپدیت خودکار لیست سرورها
- **🔄 تعادل بار**: توزیع هوشمند ترافیک بین سرورها
- **⚡ بهینه‌سازی عملکرد**: تنظیم خودکار برای بهترین نتایج

### 🛠️ **ابزارهای توسعه‌دهنده**

- **🔍 تشخیص شبکه**: مجموعه جامع عیب‌یابی

- **🔧 یکپارچگی API**: API RESTful برای خودکارسازی
- **📊 تحلیل لاگ**: تجزیه و فیلتر پیشرفته لاگ

### 🌍 **امکانات جهانی**

- **🌐 پشتیبانی چندمنطقه‌ای**: سرورها از سراسر جهان
- **🕐 آگاهی از منطقه زمانی**: زمان‌بندی و خودکارسازی هوشمند

- **🔒 انطباق**: استانداردهای امنیتی GDPR، SOC2 و سازمانی

## 🚀 شروع سریع

### 📋 **پیش‌نیازها**

- **Python 3.9+** (Python 3.12 توصیه می‌شود)
- **pip** (نصب‌کننده پکیج Python)
- **Git** (برای توسعه)

### ⚡ **نصب**

```bash
# کلون کردن مخزن
git clone https://github.com/AhmadAkd/onix-client.git
cd onix-client

# نصب وابستگی‌ها
pip install -r requirements.txt

# اجرای برنامه
python main.py
```

### 🏗️ **ساخت اجرایی**

```bash
# Windows
build.bat

# macOS/Linux
chmod +x build.sh
./build.sh

# یا از اسکریپت جهانی استفاده کنید
python build_all.py
```

## 🎨 رابط کاربری

### 🖥️ **رابط اصلی**

Onix دارای یک **رابط مدرن و بصری** با چهار بخش اصلی است:

#### 🔌 **نمای اتصال**

- **مدیریت سرور**: افزودن، ویرایش و سازماندهی سرورهای پروکسی شما
- **وضعیت بلادرنگ**: وضعیت اتصال زنده و متریک‌های عملکرد
- **عملیات سریع**: اتصال، تست و مدیریت سرورها با یک کلیک

#### 🛣️ **نمای مسیریابی**

- **قوانین هوشمند**: پیکربندی پیشرفته مسیریابی
- **لیست‌های دور زدن**: قوانین سفارشی دور زدن دامنه و IP
- **کنترل ترافیک**: مدیریت دقیق ترافیک

#### 📋 **نمای لاگ‌ها**

- **نظارت بلادرنگ**: لاگ‌های زنده برنامه و هسته
- **فیلتر پیشرفته**: جستجو و فیلتر بر اساس سطح، منبع یا محتوا
- **قابلیت‌های صادرات**: ذخیره لاگ‌ها برای تحلیل

#### ⚙️ **نمای تنظیمات**

- **ظاهر**: تم‌ها، رنگ‌ها و گزینه‌های نمایش
- **شبکه**: DNS، قوانین دور زدن و تنظیمات اتصال
- **امنیت**: Kill switch، محافظت DNS و کنترل‌های حریم خصوصی
- **پیشرفته**: تنظیم عملکرد و گزینه‌های توسعه‌دهنده

---

## 🔧 پیکربندی

### ⚙️ **تنظیمات شبکه**

```yaml
# مثال پیکربندی
network:
  mode: "rule-based"  # یا "global"
  dns_servers: ["1.1.1.1", "8.8.8.8"]
  bypass_domains: ["*.local", "*.lan"]
  tun_mode: true
  hysteria2_bandwidth:
    upload: "100 Mbps"
    download: "500 Mbps"
```

### 🛡️ **تنظیمات امنیتی**

```yaml
security:
  kill_switch: true
  dns_leak_protection: true
  firewall_integration: true
  privacy_mode: "strict"
  audit_logging: true
```
